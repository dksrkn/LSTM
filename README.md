# 🌿농작물 도매 가격 예측 프로젝트

## 🌱 프로젝트 개요
#### 프로젝트 목적 : 머신러닝을 활용하여 농작물 도매 가격을 예측한다.
#### 프로젝트 목표 : 품종 2종 각각 일주일 평균 일치율 95%

<br/>

## 👩‍💻 프로젝트 진행 과정
1. 알고리즘 선정
   - LSTM
2. 데이터
   - 도매시장 선정
   - 데이터 변수 및 출처 선정
   - 데이터 수집 및 전처리
3. 모델 학습 및 테스트
